# Personal Portfolio Website

## Description
This is my personal portfolio website built using HTML, CSS, and JavaScript. The website is fully responsive and works on both desktop and mobile screens. It showcases my skills, services, and contact information in a clean and simple design.

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Font Awesome Icons
- Google Fonts (Poppins)

## Work Done
- Created a responsive navigation bar with a mobile menu button
- Designed multiple sections like Home, About, Services, and Contact
- Used CSS variables for consistent colors and styling
- Implemented media queries for mobile responsiveness
- Added JavaScript to toggle the menu icon and navbar on small screens

## Difficulties Faced
- Mobile menu button was not opening correctly
- Icon toggle between menu and close button was confusing
- Navbar display issues in small screen sizes
- CSS and JavaScript synchronization for responsive behavior

## Solutions Learned
- Correct use of `classList.toggle()` in JavaScript
- Proper use of media queries in CSS
- Importance of correct class names and selectors
- Understanding how `display: none` and `display: block` work together

## Improvements
- Add smooth animations and transitions
- Improve UI/UX design
- Add projects section with live demos
- Convert the website into a React-based portfolio

## Conclusion
This project helped me understand responsive web design and improved my frontend development skills. It was a good learning experience for HTML, CSS, and JavaScript integration.
